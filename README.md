# 🔄 Rewrite_Code

Projeto criado com o objetivo de **revisar e reaplicar conhecimentos básicos de desenvolvimento web**, utilizando HTML, CSS, JavaScript e outras tecnologias.

---

## 📚 Sobre o projeto

O Rewrite_Code foi desenvolvido como um exercício de revisão prática, onde diferentes conceitos foram reaplicados em pequenos projetos separados.

A ideia principal foi reforçar fundamentos e melhorar a compreensão geral de desenvolvimento front-end e back-end.

---

## 🧩 Estrutura do projeto

O repositório está dividido em duas partes principais:

* **Front-end**
* **Back-end**

Cada uma dessas pastas possui seu próprio README, explicando os detalhes específicos de cada implementação.

---

## 🎨 Destaque do projeto

O principal ponto de evolução deste projeto foi o **front-end**, onde houve um foco maior em:

* Melhoria de interface
* Organização visual
* Aplicação mais clara de HTML e CSS
* Experiência de usuário mais bem estruturada

---

## 🧠 Objetivo

* Reforçar conhecimentos básicos de desenvolvimento web
* Praticar estruturação de projetos
* Melhorar interface e organização de front-end
* Revisar conceitos já estudados anteriormente
