What's HTML

Hypertext Markup Language (HTML) is the standard markup language[a] for documents designed to be displayed in a web browser

Tags

Tags HTML são instruções escritas entre colchetes angulares (< >) que definem a estrutura e o conteúdo de uma página da web, como títulos (<h1>), parágrafos (<p>) e links (<a>). A maioria das tags funciona em pares, com uma tag de abertura (ex: <p>) e uma de fechamento (ex: </p>), e instruem o navegador sobre como formatar o texto e outros elementos.

<head>
<title>
<meta>
<body>
